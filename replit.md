# Amanda's 37th Birthday Run Challenge

## Overview
A static HTML website for Amanda's 37th birthday running challenge event. The site showcases 10 running loops of 3.7 miles each, totaling 37 miles.

## Project Structure
- `docs/` - Contains the static website files
  - `index.html` - Main page with event details and route information

## Tech Stack
- Pure HTML/CSS/JavaScript (no build process)
- Google Fonts (Bebas Neue, Poppins)
- Canvas Confetti library (CDN)
- GSAP animation library (CDN)
- Embedded Google Maps iframes

## Development
The site is served using Python's built-in HTTP server on port 5000.

## Deployment
Configured as a static site deployment serving from the `docs` directory.
